# Pedra, Papel e Tesoura

Aplicativo Android desenvolvido em Java para a ATPS de Desenvolvimento Mobile - UniProjeção 2026.

## Funcionalidades
- Jogo Pedra, Papel e Tesoura contra o computador
- Geração aleatória da jogada do computador
- Placar atualizado a cada rodada
- Resultado exibido após cada jogada

## Tecnologias
- Java
- Android Studio
- API 24+
